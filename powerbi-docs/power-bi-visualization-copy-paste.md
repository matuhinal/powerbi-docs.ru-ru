# <a name="copy-and-paste-a-visualization-in-power-bi"></a>Копирование и вставка визуализации в Power BI
1. В Power BI откройте отчет в [режиме редактирования](service-reading-view-and-editing-view.md) и [создайте визуализацию](power-bi-report-add-visualizations-i.md), если в отчете еще нет этого элемента. 
2. Выберите визуализацию и используйте сочетание клавиш **CTRL + C** для копирования и **CTRL + V** для вставки.  
   ![](media/power-bi-visualization-copy-paste/copypasteviznew.gif)
